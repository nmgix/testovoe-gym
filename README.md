# Тестовое Gym

Стек:

1. React (Vite)

3. React-Router-DOM
4. Redux

## Как запустить

1. npm i
2. .env в корне с значениями

````md
VITE*API_URL = "\_base-url.app*" VITE*RATE_URL = "\_sub../list*"

```(чтобы не заиндексировалась ваша апи)
3. npm run dev
4. (опционально) в F12(девтулз) в консоли написать `dev=false`
````
